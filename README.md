# SayTextRead
## Description
This python program reads the input .txt files and produces the estimated time to finish reading.


## Requirements
Mac OS Big Sur 11.4

## 3. Usage
```
if __name__=='__main__':
    filenames=['./in_1.txt','./in_2.txt','./in_3.txt','./in_4.txt']
    saytextread(filenames,use_say=True)
```
